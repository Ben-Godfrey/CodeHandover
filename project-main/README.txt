To run our website using Node.js (which must first be installed), follow these steps:

Terminal 1:
- Go to "server" directory
- Type "node app.js"

Terminal 2:
- Go to "client" directory
- Type "npm install"
- Type "npm run dev"

Note: These instructions must be followed to view the avatar, although the website and chatbot are fully functional by simply running the index.html file.
